# Jekyll 入门教程

Jekyll 是一个静态网站生成器，不需要数据库，直接用 Markdown 写文章，非常适合个人博客、文档网站。

## 核心特点
1. 静态生成：加载速度快，无需后端维护
2. 支持 Markdown：写文章像记笔记一样简单
3. 可自定义：布局、样式都能自己改
4. 免费部署：可以部署到 Github Pages、Netlify 等平台

## 常用命令
- 新建项目：jekyll new 项目名
- 本地运行：bundle exec jekyll serve
- 构建网站：jekyll build
